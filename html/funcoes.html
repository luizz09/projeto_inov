No projeto final, quais seriam as funções?


O projeto, no final, teria diversas funções sendo elas:

- cadastro de cliente;

- cadastro de prestadores;

- sistema de pagamento;

- envio de foto/vídeo;

- avaliação;

- disputa;

- chat;

- notificação;

- mapa;

- etc;



Sistema de reputação do prestador


O aplicativo iria contar com um sistema de reputação para valorizar os prestadores que oferecem um serviço de qualidade e gerar mais confiança para os clientes.

- Níveis de classificação com estrelas, podendo ter de 0 a 5 estrelas ☆;

- Selo “Prestador Verificado”;

- Destaque para prestadores com melhores avaliações;

- Histórico público de avaliações e serviços realizados;



Modo Emergência


Para situações urgentes, o aplicativo terá a opção de modo emergência, garantindo maior prioridade no atendimento.

- Notificação prioritária para prestadores próximos;

- Maior custo para quem está pedindo o serviço;

-Indicado para casos como vazamento de água, curto-circuito ou porta trancada;



Me cite exemplos


Poderiam ser para inúmeras ocasiões diferentes que poderiam ser solicitados serviços, exemplos:

- Caso eu precisasse cortar uma árvore e não possuísse uma moto-serra, eu poderia anunciar pedindo para virem cortar as árvores da minha casa.

- Caso eu precisasse instalar um ventilador de teto e não soubesse fazer a parte elétrica, eu poderia anunciar pedindo para alguém realizar a instalação com segurança.

- Caso eu precisasse montar um computador com peças novas e não tivesse conhecimento técnico, eu poderia publicar no aplicativo solicitando ajuda para fazer a montagem.

- Caso minha pia estivesse vazando e eu não tivesse ferramentas adequadas, eu poderia anunciar o serviço para que alguém pudesse vir ajudar no reparo.



E para evitar golpes?


Para evitar fraudes foi pensado em um sistema antifraude com prova visual para garantir segurança e transparência em cada serviço. Antes do cliente pedir o serviço, ele tem que tirar fotos reais do problema para enviar ao app, assegurando que a pessoa que aceitar a tarefa saiba exatamente o que será feito. Ao chegar, ela confirma se o serviço corresponde ao do anúncio e, após a conclusão, registra um vídeo confirmando o resultado. O pagamento permanece protegido na plataforma até a confirmação final e, em caso de disputa, as provas são analisadas para um decisão justa. Assim, garantimos confiança, proteção financeira e mais segurança para todos.
